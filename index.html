User Guidelines

                
TraceTree is currently a browser-based prototype for building and viewing content tracing trees.

            

        


        

            

                

                    1. Create Root
                    
Use this when you know the original work and want to create a tracing tree for it.

                


                

                    2. Report Pirating
                    
Use this when you find suspected stolen or unauthorized content.

                


                

                    3. Automatic Tree Matching
                    
If the original content already has a tree, the report is added there. If not, a new tree is created.
