User Guidelines

1. Create Root

Any user can create a root by registering a unique piece of content.

2. Report Pirating

Anyone may report a suspected pirated copy; it becomes a gray unverified node.

3. Review / Report

Nodes can be authorized (green), marked as infringing (red), or dismissed as false.
